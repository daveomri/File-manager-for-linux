◆ main()

int main ( void  )

Main part of program lies here, user enters input command and this function redirect his demands to other functions. CController is interface between user and filesystem. This function itself allows user to change directory by typing direcory name to terminal, or shows file information by typing file name. Besides these two functions user can change mode by entering code composed with '\' + 'number' representing mode in command panel.

Returns
0 at the end of this function.
