Detailed Description

class

This class offers works directly with filesystem. It contains methods to create file, directories or links, and also methods to delete these files. This class saves files as CFile clases, also saved some of informations about these files in this class. File can be classified as directory, link or regular file. This classification is being make by three methods in this class.
